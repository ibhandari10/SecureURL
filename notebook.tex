    \subsection{Feature Engineering}\label{feature-engineering}

All we have is a string and for any model having multiple features is
important. Thus, in this section we will try to create features from the
name of our domains. Since XGBoost needs numeric features, we need to
encode the characters. I have created a dictionary of our character
